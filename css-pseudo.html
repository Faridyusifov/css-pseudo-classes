:hover, :focus, :active pseudo classes ne vaxt tetbiq olunur

Ne vaxt :hover tetbiq olunur

Ne vaxt :focus tetbiq olunur

Ne vaxt :active tetbiq olunur

Her 3 pseudo tetbiq-i

Hover

:hover tetbiq etdiyimizde meselen sicani hemin elementin uzerine getirdikde bas verecek deyisikleri yaza bilerik

button:hover{
    background-color : green ;
}

Meselen sicani asagidaki button-un uzerine getirdikde reng yashila deyishecek

Focus

:focus tetbiq etdiyimizde meselen sicani hemin elementin uzerine vurduqda deyisikliyi verirsen.Bu deyisiklik ta ki ekranda bashqa yere vurana qeder bash verir.

button:focus{
   background-color : yellow ;
}

Meselen sicani asagidaki button-un uzerine getirib click-lesek reng sari renge deyisecek ve ekranda bashqa yere click-liyene qeder sari olacaq.

Active

:active tetbiq etdiyimizde meselen sicani hemin elementin uzerine getirib aktivleshdirdikde bash veren deyishikliyi veririk 

button:active{
   background-color : red ;
}

Meselen sicani asagidaki button-un uzerine getirib aktivleshdirdikde qirmizi reng olacaq.

Her 3 pseudo class-larini bu sira ile yazmaq lazimdi

Melumdurki user seyfede ilk olaraq sicani birinci elementin uzerine getirecey,sonra fokuslayib active edecey.

button:hover{
   background-color : green ;
}

button:focus{
    background-color : yellow ;
}

 button:active{
    background-color : red ;
}
